# lri-proto

Protobuf definitions and generated Rust types for the Light L16 LRI format.

Derived from [dllu/lri-rs](https://github.com/dllu/lri-rs) (protobuf extracted from Lumen). Gently modified for this workspace.

See [LRI.md](../LRI.md) for how messages map to the parser in `lri-rs`.
